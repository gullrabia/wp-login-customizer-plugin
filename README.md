# WP Login Customizer Plugin

A simple and lightweight WordPress plugin that allows you to customize the WordPress login page easily. This plugin is useful for developers and site owners who want to personalize the login experience without touching core files.

---

##  Features

- Customize the WordPress login page layout  
- Modify login form styles  
- Clean and developer-friendly code structure  
- Easy to extend for future features  
- No core file changes required
